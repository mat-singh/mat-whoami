Mat API: Timestamp microservice

What do I do:
    1) If you pass a string as a parameter, I will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
    2) If it does, I will return both the Unix timestamp and the natural language form of that date.
    3) If it does not contain a date or Unix timestamp, I will return null for both properties.

Example usage:

https://mat-timestamp.glitch.me/January 10, 2018
https://mat-timestamp.glitch.me/1515542400

Example output:

{ "unix": 1515542400, "natural": "January 10, 2018" }
